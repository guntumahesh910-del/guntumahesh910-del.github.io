What is HTML?

HTML stands for HyperText Markup Language. It is used to create the structure of websites.

Basic HTML Structure

This is a simple HTML structure:

<!DOCTYPE html>
<html>
  <head>
    <title>My Website</title>
  </head>
  <body>
    Hello World
  </body>
</html>
